# PROJECT NEO — Frontend

React 19 + TypeScript + Vite frontend for the PROJECT NEO trading terminal.

See the [root README](../README.md) for full project documentation, setup instructions, and deployment guide.

## Quick Start

```bash
npm install
npm run dev       # http://localhost:5173 (proxies /api → localhost:8000)
npm run build     # production build → dist/
```
